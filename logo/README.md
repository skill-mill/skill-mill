# Logo

Logo assets for the skill mill organization.

## Font

[Audiowide](https://fonts.google.com/specimen/Audiowide) (SIL Open Font License 1.1)

## Generate

```bash
cd logo
npm install
node generate-logo.mjs
```

Outputs SVG and PNG (1364x1364px) in 4 variants:

| File | Background | Text |
|------|-----------|------|
| skill-mill-light | White | Black |
| skill-mill-dark | Black | White |
| skill-mill-black | Transparent | Black |
| skill-mill-white | Transparent | White |
